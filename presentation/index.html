Sum of jsons

Input

                
                {"a": 1, "b": 2}
                {"a": 3, "b": 4}

Output 

                
                    {"a": 4, "b": 6}

Solution

                
                SELECT jsonb_object_agg(key, val)
                FROM (
                    SELECT key, SUM(value::numeric) val
                    FROM test_table t, jsonb_each_text(info)
                    GROUP BY key
                 )s;
